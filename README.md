## 💁 웹 서비스 소개
---

**ALTA**는 알고리즘 스터디의 기록을 도와주는 서비스입니다.

레포지토리를 생성하고 관리하는 부분을 자동화하여 알고리즘 문제풀이에만 집중할 수 있도록 해줍니다.

[데모 영상]()

<br />

> 서비스를 이용하려면 Github 계정이 있어야 하며, 연동을 필요로 합니다.
